# SemanticFeed

SemanticFeed is an AI system for managing information overload.

It helps users save articles, PDFs, notes, and links, then summarizes, classifies, ranks, and retrieves them using semantic search and RAG.

## Core Features

- URL ingestion
- PDF ingestion
- Content summarization
- Topic classification
- Difficulty detection
- Reading time estimation
- Semantic search
- Priority ranking
- RAG-based question answering

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- pgvector
- Redis
- Docker
- SQLAlchemy
- Pydantic
